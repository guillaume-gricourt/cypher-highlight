# cypher-latex

[![Build Status](https://travis-ci.org/szarnyasg/cypher-highlight.svg?branch=master)](https://travis-ci.org/szarnyasg/cypher-highlight)

To run all three LaTeX compilers, use:

```
make pdflatex xelatex lualatex
```
